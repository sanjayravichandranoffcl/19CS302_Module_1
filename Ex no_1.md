# EX 1 C program to initialize the value as 5.8 & display the same.
## DATE:
## AIM:
To write a program to initialize the value as 5.8 & display the same.

## Algorithm

1. Start the program.
2. Declare a floating-point variable `a`.
3. Initialize the variable `a` with the value **5.8**.
4. Display the value of `a` using `printf()`.
5. Stop the program.

## Program:
```
#include <stdio.h>
int main()
{
    float a=5.8;
    printf("%f",a);
    return 0;
}
```
